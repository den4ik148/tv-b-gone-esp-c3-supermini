🇺🇸 English VersionESP32-C3 Super Mini TV-B-GoneA compact, powerful TV-killer based on the ESP32-C3 Super Mini. This project allows you to turn off almost any TV by cycling through hundreds of IR codes from different manufacturers (EU/NA regions).⚡ FeaturesMassive Database: Includes all classic TV-B-Gone codes (v1.2) for Europe and North America.Ultra Compact: Specifically tuned for the tiny ESP32-C3 Super Mini form factor.Smart Control: Uses the built-in BOOT button to start/stop the attack.SpecOS Core: Optimized bit-banging for IR signal generation.🔌 WiringComponentESP32-C3 PinNoteIR LEDGPIO 6Use a transistor (2N2222/BC547) + resistor!ButtonGPIO 5 / 9GPIO 9 is the default BOOT button.🛠 How to useInstall VS Code + PlatformIO.Clone this repo.Flash the firmware to your ESP32-C3.Open Serial Monitor (115200) to see the progress.Point at a TV and press the button.
